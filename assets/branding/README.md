# Branding — logo (tú lo diseñas)

Deja aquí el logo del proyecto. **Nadie del equipo de agentes debe generar ni sobrescribir tu diseño.**

## Archivos esperados

| Archivo | Uso |
|---------|-----|
| `logo.png` | Principal (README, demo, Devpost) — recomendado |
| `logo.svg` | Vector opcional |
| `logo-dark.png` | Opcional, fondos oscuros |
| `favicon.png` | Opcional 32×32 o 64×64 |
| `og-cover.png` | Opcional 1200×630 para redes / Devpost |

## Sugerencias rápidas (no obligatorias)

- Nombre producto: **NertzMetalEngine** / **_Metrics_**
- Formatos: PNG con transparencia o SVG
- Tamaño útil logo: 512×512 o 1024×1024
- Mantén margen; evita texto microscópico (video demo &lt; 3 min)

## Cómo se usará

1. README / carpeta del repo  
2. Video demo del hackathon  
3. Submission en [openai.devpost.com](https://openai.devpost.com/)

Cuando termines, solo guarda el archivo aquí con el nombre `logo.png` (o `.svg`).
